The Most Famous Musicians You've Never Heard of

This project aims to identify popular artists outside of the Anglospehere, and understand global music trends more broadly. Using data scraped from a variety of sources, including Spotify chart data and public opinion data, Sarah Helmbrecht and I created this Jupyter notebook data science project that includes ETL, visualisations, and an implementation of a random forest machine learning model.
